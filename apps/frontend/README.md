# Frontend (Vite + React)

Short, project-specific frontend README. For full project context, see the root `README.md`.

## Setup
From repo root:
```bash
pnpm install
```

From this folder:
```bash
cp .env.example .env
pnpm dev
```

Frontend: http://localhost:5173
